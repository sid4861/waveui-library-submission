Avatar

The avatar represents a user, and displays the profile picture, initials or fallback icon. It comes in three sizes, small, medium, large

Usage and examples

Just use the img tag with the appropriate class

Small Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-small">

Medium Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-medium">

Large Avatar

                
                    <img src="https://bit.ly/dan-abramov" alt="avatar" class="avatar-large">

Alert

Use the div tag with the approprate alert class. There are 5 types of alert classes available - primary, success, error, warning, and info

Button

Wave UI 4 offers types of buttons - Primary, Link, Icon, Floating Action Button

Usage and examples

You the button tag with appropriate class

Primary Button

Use can use color classes of warning, info, success and error as well

                
                    <button class="button button-primary">Primary</button>
                    
                    <button class="button button-warning">Warning</button>

Link Button

You can use color classes of warning, info, success and error as well

                 
                     <button class="button button-link-primary">Primary</button>
                     
                     <button class="button button-link-warning">Warning</button>

Icon Button

surround an icon with a button tag.

                 
                     <button class=" button-icon-primary">P</button>
                     
                     <button class=" button-icon-warning"<W</button>

Floating Action Button

You can use different color classes

                 
                     <button class=" button button-floating-primary">P</button>
                     
                     <button class="button button-floating-warning"<W</button>

                    src="https://images.unsplash.com/" alt="" class="image">
